# myFinancialTracker Website

This is a multipage website for the myFinancialTracker iOS app. The website is built as a Single Page Application (SPA) using HTML5, CSS, and vanilla JavaScript.

## Features

- Responsive design for desktop and mobile devices
- Smooth page transitions
- Interactive slideshows on multiple pages
- Login and account functionality
- Subscription management system

## Pages

1. **Home Page** - Overview of the app with feature highlights
2. **Features Page** - Detailed information about the app's features
3. **Support & Suggestions Page** - Contact form for support requests
4. **Testimonials Page** - User testimonials in a slideshow format
5. **Sign Up Page** - Subscription options and signup form
6. **Account Page** - User account management (visible after login)

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser

## Structure

```
myFinancialTracker-Website/
├── index.html           # Main HTML file
├── css/
│   └── styles.css       # Styles for the website
├── js/
│   └── app.js           # JavaScript functionality
└── images/              # Images used in the website
```

## Development

This project uses vanilla JavaScript without any frameworks or build tools. You can modify the files directly and refresh the browser to see changes.

## Note

This is a frontend demonstration only. In a production environment, you would need to implement proper backend services for:

- User authentication
- Data storage
- Payment processing
- Form submission handling
